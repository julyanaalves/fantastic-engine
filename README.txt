============================
EXPERIÊNCIA INTERATIVA
============================

Objetivo:
Completar 3 minigames em sequência (pontes, organizar formas, crescer a semente) dentro do tempo.

Arquivos necessários (devem estar JUNTOS na MESMA pasta):
- index.html
- sketch.js
- style.css

Como abrir:
1. Certifique-se de que os três arquivos acima estão na mesma pasta.
2. Abra o arquivo index.html diretamente em um navegador moderno (Chrome, Firefox, Edge, Safari).
   - Dê duplo clique em index.html OU
   - Clique com botão direito > Abrir com > seu navegador
3. A tela inicial será exibida. Clique em "Iniciar" para começar.

Se o navegador bloquear algo:
- Verifique se não há extensões interrompendo scripts.
- Caso algo não apareça, tente outro navegador.

Controles rápidos:
Nível 1 (Pontes): Clique para adicionar elos. Backspace remove o último elo. R reinicia.
Nível 2 (Formas): Arraste cada forma para a zona correspondente. R reinicia.
Nível 3 (Semente): Mova o mouse para coletar orbes e crescer a semente. R reinicia.

Tempo:
Cada nível tem limite de 20 segundos. A barra no topo mostra o tempo restante.

Problemas comuns:
- Nada acontece ao clicar: Talvez você não tenha clicado em "Iniciar" na tela inicial.
- Undo não funciona: Só existe no Nível 1 e precisa haver pelo menos um elo.
- Tela travada: Abra o console (F12) e veja se há erros.

Requisitos:
- Apenas um navegador moderno. Sem necessidade de servidor local.
- Se quiser rodar via servidor (opcional), pode usar por exemplo:
  python -m http.server 8080
  e acessar http://localhost:8080
